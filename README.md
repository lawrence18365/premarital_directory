# Wedding Counselors

https://weddingcounselors.com

React-based directory connecting engaged couples with premarital counselors.

## Quick Start
```bash
cd client
npm install
npm start
```

## Structure
- `/client` - React app
- `/supabase` - Database
- `/scripts` - Utilities (organized by function)
- `/docs` - Documentation

See `/docs/SOFT_404_FIX_IMPLEMENTATION.md` for recent SEO fixes.
